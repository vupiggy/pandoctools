Code Snippet Example

type Figure struct {
	Target	string
	Path	string		`json:path`
	Caption	string		`json:caption`
	Label	string		`json:label`
	Options	string		`json:options`
	Place	string		`json:place`
	Suffix  map[string]string
}

const latexTemplate = `
\begin{figure}[{{.Place}}]
  \centering
  \includegraphics[{{.Options}}]%
      {{"{"}}{{.Path}}.{{index .Suffix .Target}}{{"}"}}
  \caption{{"{"}}{{.Caption}}{{"}"}}
  \label{{"{fig:"}}{{.Label}}{{"}"}}
\end{figure}
`

amsthm Example

Circle

(Plane) In mathematics, a plane is a flat, two-dimensional surface that extends infinitely far.
(Circle) A circle is a shape consisting of all points in a plane that are a given distance from a given point, the centre; equivalently it is the curve traced out by a point that moves in a plane so that its distance from a given point is constant.

Chord and tangent line

(Chord) A line segment whose endpoints lie on the circle, thus dividing a circle in two sements.
(Tangent line) A tangent line to a circle is a line that touches the circle at exactly one point, never entering the circle's interior.
